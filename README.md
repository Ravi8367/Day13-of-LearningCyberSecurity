# Damn Vulnerablen Web Application

DVWA is a PHP/MySQL web application intentionally built with numerous security flaws, designed as a safe, legal platform for security professionals, developers, and students to practice, learn, and test web application security skills, including identifying vulnerabilities like SQL Injection, Cross-Site Scripting (XSS), and Command Injection.

# DVWA Installation

1.Install XAMPP Server - Download and install.

2.Launch XAMPP Control Panel

3.Start Apache and MySQL

4.Download DVWA and Extract to xampp/htdocs.

5.Configure DVWA: Rename

6.config.inc.php.dist to config.inc.php

7.Access DVWA: Type localhost in browser

8.Login: userename - admin, password - password
